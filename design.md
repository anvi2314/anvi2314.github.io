# Anvita's Corner – Personal Portfolio Website

## Project Overview
A playful, colorful personal website that introduces Anvita, showcasing hobbies, interests, and achievements. The site is designed to be welcoming, fun, and easy to navigate for classmates, family, and teachers.

## Purpose & Goals
- **Purpose:** Introduce Anvita's personality, hobbies, and interests in an engaging way.
- **Goals:**
  - Visitors can learn key facts about Anvita in under 2 minutes.
  - Content is age-appropriate, fun, and visually appealing.
  - Navigation is intuitive on desktop and mobile devices.

## Target Audience
- Classmates, teachers, and family members
- Visitors aged 8–18

## Visual Design

### Color Palette
- **Primary:** Bright, vibrant color (e.g., turquoise, coral, or electric purple)
- **Secondary:** Complementary pastel (soft pink, mint, or light yellow)
- **Background:** Light or neutral (white, light gray, or soft cream)
- **Accent:** Energetic color for buttons and highlights

### Typography
- **Headings:** Rounded sans-serif font (fun, modern, friendly)
- **Body:** Clean, readable sans-serif (e.g., Segoe UI, Open Sans)
- **Font Size:** Large enough for comfortable reading (16px+ body text)

### Design Style
- Playful and kid-friendly
- Colorful icons and simple illustrations
- Rounded corners on cards and buttons
- Ample whitespace for clarity

### Background & Imagery
- **Background Images/Patterns:** Fun, age-appropriate visuals (e.g., subtle geometric patterns, stars, cricket bats, musical notes, robots, colorful doodles).
- **Image Style:** Hand-drawn or cartoon illustrations; avoid overly realistic or mature imagery.
- **Suggested Motifs:** Animals (tigers, birds), sports equipment (cricket bat, ball), hobbies (piano keys, code symbols), nature (mountains, clouds), and playful geometric shapes.
- **Image Placement:** Use as subtle background textures or section dividers; keep main content readable and not overshadowed.

## Animated Elements
- **Cricket Animation:** Animated illustration of a cricket batter in action (can be SVG or CSS-based).
  - Shows dynamic movement (swing, ball trajectory).
  - Light, engaging, non-distracting.

## Accessibility & Privacy

### Accessibility
- High color contrast (WCAG AA compliant minimum).
- Keyboard navigation support (Tab, Enter, Escape).
- Alt text on all images and illustrations.
- Semantic HTML (headings, lists, buttons properly marked).

### Privacy & Safety
- No personal contact information displayed (email, phone, full address).
- No real photos of people outside of Anvita (when appropriate).
- Use generic or illustrated images for Smriti Mandhana reference instead of real photos (unless licensed).

## Technical Stack

### Architecture
- **Type:** Static website with optional animations.
- **Hosting:** GitHub Pages.

### Technologies
- **HTML5:** Semantic markup, responsive layout.
- **CSS3:** Flexbox/Grid, transitions, animations.
- **JavaScript (minimal):** Smooth scrolling, interactive elements if needed.
- **Animation:** CSS keyframes or lightweight SVG animation; consider Lottie for complex sequences.

### Performance
- Optimize images (compress, use WebP format where possible).
- Minimize CSS/JS bundles.
- Mobile-first responsive design.
